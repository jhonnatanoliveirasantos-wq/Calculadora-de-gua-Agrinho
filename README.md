# Calculadora de Desperdício de Água na Irrigação 🌾💧
Este projeto foi desenvolvido para o Concurso Agrinho 2026 (Subcategoria Programação).
## 📋 Sobre o Projeto
O objetivo é ajudar o pequeno produtor rural a calcular a quantidade de água que pode economizar,mudando o sistema tradicional de irrigação para o gotejamento, aliando tecnologia com preservação ambiental.
## ♿ Acessibilidade
- **Alto Contraste:** Facilita a leitura de pessoas com baixa visão.
- **Aumento de Fonte:** Permite o aumento do tamanho do texto dinamicamente.
- **HTML Semântico:** Facilita o uso de leitores de ecrã.
## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (JS)
